# REASONS Canvas Notes

The REASONS Canvas reference exists to keep prompt-generation terminology and structure consistent across the plugin.

## Why it matters

`spdd-reasons-canvas`, `spdd-generate`, `spdd-prompt-update`, and `spdd-sync` all depend on prompt structure staying predictable.

The shared reference helps:

- align prompt structure across phases
- keep terminology stable
- avoid each skill redefining the same concepts differently

For the authoritative repository source, see `plugins/specark/references/reasons-canvas.md`.
